<TSNotes>
  The type of the value passed to the `prepareValue` function is `FieldValue`.
  This is a union of the various types that fields values can be, including
  strings, booleans, objects, etc.  In this example we use the `String()`
  function to cast it to a string.  It's strictly unnecessary in this case
  because we know it's a text field and the value will always be a string.
  But this keeps Typescript happy by telling it that we know what we're doing.
</TSNotes>
