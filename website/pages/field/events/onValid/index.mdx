## `onValid`

<Split>
  You can add an `onValid` handler to a <FieldLink/> which will be called
  when the field is successfully validated. The field context will be passed
  as an argument.

  <TryItOut>
    Enter some text in one of the fields below and then select the other
    field.  The `validateOnBlur` option on the <FormLink/>
    means that the first field will be validated when it loses focus and
    the `onValid` handler will be called.
  </TryItOut>
</Split>

<TSNotes>
The `onValid` function has a type of `FieldContextFunction` and receives
the field context as the first argument with a type of `FieldContextItems`.
</TSNotes>
