## `onLoad`

<Split>
  You can add an `onInvalid` handler to a <FieldLink/> which will be called
  when the field is initialised.  This is useful if you want to
  programmatically control the field.
  <TryItOut>
    Click on the buttons to programmatically call field methods.
    The first resets the field, the next two set the field value, and
    the last one focusses the field.
  </TryItOut>
</Split>
