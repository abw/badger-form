## `prefix`

You can add a `prefix` to a `text` field or any
of the other types using the basic text input, e.g.
`money`, `date`, etc.  The prefix can be text,
an SVG icon or any other JSX fragment.

The `prefixClass` property can be used to set a CSS class
for the prefix.  Badger CSS provides two built in styles:
`lined` to add a dividing line, and `shaded` to
add some background shading.
