## `useForm`

If you want to set the values from a component placed inside the
`Form` component then you can use `useForm()` to
get access to the form context.

## Preventing Default Event

If you're using a `button` to trigger the call to
`setValues()`, as shown in the example above, then you need
to make sure you call `e.preventDefault()` event.  Otherwise
the form may be submitted. Alternately you can pass the event as the
second argument to the `setValues()` method as shown below
and it will do it for you.
