# Input Types

## `checkbox`

Set the <FieldLink/> <FieldTypeLink/> to `checkbox` for a checkbox.
Use the `text` property to define any text that you want to
appear alongside the checkbox.  The `label` is optional on all
fields, so depending on what your checkbox text says, you might want to
leave it out.
