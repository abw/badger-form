## Using `addInput()`

The other approach is to use `addInputType()` to register your component. You
can then use the <FieldTypeLink/> property to access it using the name you
give it. In this example we give it the name `colorButton`.

Note that input type names are converted to a canonical form, both when being
defined and being accessed. Non-word characters are removed and the name is
converted to lower case. As shown in this example, input types of
`colorButton`, `color-button` and `colorbutton` are all considered to be the
same thing.
