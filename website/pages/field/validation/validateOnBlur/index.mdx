## `validateOnBlur`
<Split>
  The `validateOnBlur` property can be set to trigger validation whenever a
  field is blurred.  You can set it on individual <FieldLink/> components or
  on the parent <FormLink/> to have it apply to all fields.

  <TryItOut>
    Try switching focus from one field to another without entering
    anything.  You should see the custom validation errors when each
    field is blurred.
