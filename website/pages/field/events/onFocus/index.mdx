## `onFocus`

<Split>
  You can add an `onFocus` handler to a <FieldLink/> which will be called
  when the field is focussed (i.e. selected). The field context will be
  passed as an argument.

  <TryItOut>
    Focus on each of the fields below.  A message should be displayed
    each time a field gains focus.
  </TryItOut>
</Split>

<TSNotes>
The `onFocus` function has a type of `FieldContextFunction` and receives
the field context as the first argument with a type of `FieldContextItems`.
</TSNotes>
