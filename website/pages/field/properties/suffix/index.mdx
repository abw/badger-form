## `suffix`

The `suffix` property can be used to a a suffix to a field.
This can be text, an SVG icon or other JSX fragment.

<Example
  Component={Suffix}
  code={SuffixSrc}
  caption="suffix"
  highlightLines="9"
  expand
/>

The `suffixClass` property can be used to set a CSS class
for the suffix.  Badger CSS provides two built in styles: {' '}
`lined` to add a dividing line, and `shaded` to
add some background shading.
