## `fields`

The `fields` property can be used as a shortcut to render a set of named
fields using the <FieldsLink/> component.

## `grid`, `gap` and `stack`

Any other properties are forwarded onto the <FieldsLink/> component. For
example, you can set the `grid`, `gap` and `stack` properties to set the
layout to implement a grid layout.
