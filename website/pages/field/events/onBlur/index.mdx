## `onBlur`
<Split>
  You can add an `onBlur` handler to a <FieldLink/> which will be called when
  the field is blurred (i.e. loses focus). The field context will be passed
  as an argument.

  <TryItOut>
    Focus on one of the fields below and then select the other field.
    A message should be displayed each time a field loses focus.
  </TryItOut>
</Split>

/>


<TSNotes>
The `onBlur` function has a type of `FieldContextFunction` and receives
the field context as the first argument with a type of `FieldContextItems`.
</TSNotes>
